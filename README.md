## 🚀 主要專案

### 台鐵車站資訊查詢系統
**位置**: `practice/taiwan-railway-project/`

這是一個完整的 Python GUI 應用程式，展示了現代軟體開發的最佳實踐：

#### 🌟 專案特色
- **完整的 MVC 架構**：清楚分離模型、視圖和控制器
- **資料庫整合**：PostgreSQL/SQLite 支援
- **現代化 GUI**：使用 Tkinter 開發的使用者友善介面
- **完整測試覆蓋**：單元測試、整合測試、端到端測試
- **規格驅動開發**：完整的需求分析和系統設計文件

#### 🎯 主要功能
- 車站搜尋和資訊查詢
- 客流量統計和分析
- 多車站比較功能
- 圖表視覺化和資料匯出

#### 🚀 快速開始
```bash
cd practice/taiwan-railway-project
python main.py
```
